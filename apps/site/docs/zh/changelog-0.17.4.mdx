# v0.17.4 更新日志：让 AI 看见页面 DOM！

## 数据查询 API 全面增强

为满足更多前端自动化和数据提取场景，以下 API 新增了 options 参数，支持更灵活的 DOM 信息和截图传递：

- `agent.aiQuery`
- `agent.aiBoolean`
- `agent.aiNumber`
- `agent.aiString`

### 新增 options 参数

- `domIncluded`：是否传递精简后的 DOM 信息（适用于提取隐藏属性，如图片链接等），默认关闭。
- `screenshotIncluded`：是否传递截图，默认开启。

### 代码示例

```typescript
// 提取通讯录中所有联系人的完整信息（包含隐藏的头像链接）
const contactsData = await agent.aiQuery(
  "{name: string, id: number, company: string, department: string, avatarUrl: string}[], extract all contact information including hidden avatarUrl attributes",
  { domIncluded: true }
);

// 检查通讯录中第一个联系人的 id 属性是否为 1
const isId1 = await agent.aiBoolean(
  "Is the first contact's id is 1?",
  { domIncluded: true }
);

// 获取第一个联系人的 ID（隐藏属性）
const firstContactId = await agent.aiNumber("First contact's id?", { domIncluded: true });

// 获取第一个联系人的头像 URL（页面上不可见的属性）
const avatarUrl = await agent.aiString(
  "What is the Avatar URL of the first contact?",
  { domIncluded: true }
);
```

## 新增右键点击能力

你有没有遇到过需要自动化右键操作的场景？现在，Midscene 支持了全新的 `agent.aiRightClick()` 方法！

- **功能**：自动右键点击页面元素，适用于那些自定义了右键事件的场景（注意：无法与浏览器原生菜单交互）。
- **真实场景示例**：
  ```typescript
  // 在通讯录应用中右键点击联系人，触发自定义上下文菜单
  await agent.aiRightClick("Alice Johnson", { deepThink: true });
  
  // 然后可以点击菜单中的选项
  await agent.aiTap("Copy Info"); // 复制联系人信息到剪贴板
  ```
- **参数说明**：
  - `locate`: 用自然语言描述你要操作的元素
  - `options`: 可选，支持 `deepThink`（AI精细定位）、`cacheable`（结果缓存）

### 示例代码

```typescript
import puppeteer from "puppeteer";
import os from "node:os";
import { PuppeteerAgent } from "@midscene/web/puppeteer";
import "dotenv/config"; // read environment variables from .env file

const sleep = (ms) => new Promise((r) => setTimeout(r, ms));

Promise.resolve(
  (async () => {
    const browser = await puppeteer.launch({
      headless: false, // set to 'false' to see the browser window for demo
      args: ["--no-sandbox", "--disable-setuid-sandbox"],
    });

    const page = await browser.newPage();
    await page.setViewport({
      width: 1280,
      height: 768,
      deviceScaleFactor: os.platform() === "darwin" ? 2 : 1,
    });

    // Load the contacts demo page (replace with your actual file path or URL)
    await page.goto("https://lf3-static.bytednsdoc.com/obj/eden-cn/nupipfups/Midscene/contacts3.html");
    
    // await sleep(2000);

    // 🤖 Initialize Midscene agent
    const agent = new PuppeteerAgent(page);

    console.log("🚀 Starting Smart Contacts Demo with Midscene AI");
    console.log("================================================");

    // ✨ 新特性示例 1: aiRightClick - 右键点击联系人
    console.log("\n1. 🖱️ 测试 aiRightClick 功能...");
    await agent.aiRightClick("Alice Johnson", { deepThink: true });
    await sleep(1000);
    console.log("✅ 成功右键点击 Alice Johnson 的联系人卡片");
    
    // Click on "Copy Info" option in context menu
    await agent.aiTap("Copy Info");
    await sleep(1000);
    console.log("✅ 成功触发 'Copy Info' 操作");

    // ✨ 新特性示例 2: aiQuery with domIncluded - 提取包含隐藏属性的联系人数据
    console.log("\n2. 📊 测试 aiQuery with domIncluded 功能...");
    const contactsData = await agent.aiQuery(
      "{name: string, id: number, company: string, department: string, avatarUrl: string}[], 提取所有联系人信息，包括隐藏的 avatarUrl 属性",
      { domIncluded: true }
    );
    console.log("✅ 成功提取包含隐藏属性的联系人数据:");
    console.log(JSON.stringify(contactsData, null, 2));

    // ✨ 新特性示例 3: aiBoolean with domIncluded - 检查 ID 字段
    console.log("\n3. ❓ 测试 aiBoolean with domIncluded 功能...");
    const isId1 = await agent.aiBoolean(
      "第一个联系人的 id 是否为 1?",
      { domIncluded: true }
    );
    console.log("✅ 第一个联系人的 id 是否为 1?", isId1);

    // ✨ 新特性示例 4: aiNumber - with domIncluded - 统计联系人数量
    console.log("\n4. 🔢 测试 aiNumber with domIncluded 功能...");
    const contactCount = await agent.aiNumber("第一个联系人的 id?", { domIncluded: true });
    console.log("✅ 第一个联系人的 id:", contactCount);

    // ✨ 新特性示例 5: aiString with domIncluded - 获取第一个联系人的 ID
      console.log("\n5. 🆔 测试 aiString with domIncluded 功能...");
    const firstContactId = await agent.aiString(
      "第一个联系人的头像 URL 是什么?",
      { domIncluded: true }
    );
    console.log("✅ 第一个联系人的头像 URL:", firstContactId);

    console.log("\n🎉 智能通讯录演示完成!");
    console.log("================================================");
    console.log("✨ 新 Midscene 特性示例:");
    console.log("   • aiRightClick() with deepThink - 自定义上下文菜单");
    console.log("   • aiQuery() with domIncluded - 提取隐藏 ID 属性");
    console.log("   • aiBoolean() with domIncluded - DOM 基于布尔检查");
    console.log("   • aiNumber() - with domIncluded - 隐藏 ID 属性");
    console.log("   • aiString() with domIncluded - 提取隐藏头像 URL 值");

    // 保持浏览器打开几秒钟以查看结果
    await sleep(3000);
    await browser.close();
  })()
);
```
