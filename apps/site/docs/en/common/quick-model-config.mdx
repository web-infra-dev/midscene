Midscene drives the UI with a multimodal model. Set these environment variables to get started. The example below uses **Qwen3-VL** via OpenRouter — it is easy to obtain and a solid default:

```bash
export MIDSCENE_MODEL_BASE_URL="https://openrouter.ai/api/v1"
export MIDSCENE_MODEL_API_KEY="your-openrouter-api-key"
export MIDSCENE_MODEL_NAME="qwen/qwen3-vl-235b-a22b-instruct"
export MIDSCENE_MODEL_FAMILY="qwen3-vl"
```
