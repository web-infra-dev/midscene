## Midscene Android World Benchmark

Run [Android World benchmark](https://github.com/google-research/android_world) using Midscene.


## Contents

This package contains:

* TypeScript Android World Benchmark runner, ported from [Android World benchmark](https://github.com/google-research/android_world).
* All benchmark scenes in [Android World benchmark](https://github.com/google-research/android_world) , written in TypeScript
